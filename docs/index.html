ClubCalendar

Enhanced Event Calendar Widget for Wild Apricot

Get Started View on GitHub

Why ClubCalendar?

Smart Filtering

Let visitors filter events by activity type, time of day, availability, and custom categories. Find the right event in seconds.

Auto-Tagging

Automatically categorize events based on naming patterns. "Happy Hikers: Trail Walk" gets tagged as a hiking event automatically.

Modern Design

Clean, responsive calendar that looks great on desktop and mobile. Customizable colors to match your organization's branding.

Easy Setup

Simple embed code works on any Wild Apricot page. No coding skills required for basic setup.

Read-Only & Safe

ClubCalendar never modifies your Wild Apricot data. It only reads events and displays them.

Free Hosting Options

Deploy on Google Cloud's free tier or your own server. No ongoing costs for most organizations.

How It Works

┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│  Wild Apricot   │      │   Sync Job      │      │   Your Site     │
│    Events       │ ───► │  (every 15 min) │ ───► │   Calendar      │
│                 │      │                 │      │   Widget        │
└─────────────────┘      └─────────────────┘      └─────────────────┘
1

Sync Job Runs

Every 15 minutes, a sync job fetches your events from the Wild Apricot API

2

Events Transformed

Events are enhanced with auto-tags and saved as a JSON file

3

Widget Loads Data

The calendar widget fetches the JSON and displays your events

4

Visitors Browse

Site visitors filter and explore events without touching WA data

Embed Code Example

Add this code to any Wild Apricot page using an HTML/Code gadget: 

<!-- ClubCalendar Widget -->
<div id="clubcalendar"></div>
<script>
window.CLUBCALENDAR_CONFIG = {
    eventsUrl: 'https://yourserver.com/data/events.json',
    title: 'Find Events',
    primaryColor: '#2c5aa0'
};
</script>
<script src="https://yourserver.com/widget/clubcalendar-widget.js"></script>

Requirements

Wild Apricot

Admin access to create API credentials and edit pages. Must whitelist external JavaScript domain.

For Custom Server

Linux server with Python 3.8+, cron, and a web server (Apache/Nginx).

For Google Cloud

Google account. Uses free tier: Cloud Functions, Cloud Scheduler, Cloud Storage.
